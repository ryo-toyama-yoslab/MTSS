◇ 概要 ◇
筋トレ初心者を対象としている。筋トレ初心者はWebを見ても情報量が多いので、自分に合った筋トレ方法を選ぶの
のが難しク、筋トレを継続することが難しいと我々は考えた。そこで、我々は、筋トレ初心者が
簡単に適切な筋トレ方法を探すことができ、筋トレのモチベーションを維持をすることができるアプリを開発した。

◇ 動作条件 ◇
rawファイル、drawableファイルが必須。Android端末内で動作可能

◇ ファイル構成 ◇
app
 -build
 -src
  -main
   -java 
   -res
    -font
    -layout
    -drawable
    -drawable-24
    -raw
    -value


◇ インストール ◇
PCにAndroid端末をUSBケーブルにつないでインストールする。

◇ アンインストール ◇
展開してできたフォルダをまるごと削除すれば、アンインストール完了です。レジストリは一切変更していません。


◇ つかいかた ◇
  ●つかいかた
設定画面・・・男女選択、モード選択、ダンベルの重量選択
部位選択画面・・・筋トレしたい部位をタップすると、その部位の解説動画の選択画面へと遷移する。
メニュー画面・・・おススメの筋トレ動画を表示する。
カレンダー画面・・・その日した筋トレ名や回数を記録する。
動画選択画面・・・初級、中級、上級でレベル分けされており、キーとなる筋トレ動画を3回見ると次のレベルへと進むことができる。セット回数を入力することで、カレンダー画面に反映される。
